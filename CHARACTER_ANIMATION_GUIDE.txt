- Add Component > PlayerController (for movement)

- Project > Create > Animator Controller > "Auron_Controller"
- Auron > Animator > Controller > Assign "Auron_Controller"

STEP 5: SETUP ANIMATIONS
- Window > Animation > Animation
- Create clips: Idle, Walk, Attack, Skill1, Skill2
