-------------------------------------------------------------------
RESPONSIVE DESIGN NOTES
-------------------------------------------------------------------

PORTRAIT MODE (9:16, 1080x1920):
- Joystick: Bottom left, unchanged
- Skills: Bottom right, unchanged
- Top bars: Compressed, currency icons only (no text)
- Game viewport: Taller, more vertical space

LANDSCAPE MODE (16:9, 1920x1080):
- Default layout as shown above
- More horizontal space for UI elements

NOTCH HANDLING:
- Safe area insets: 44px top (iPhone X+), 0px bottom
- Move top UI down if notch detected
- Use Unity's Screen.safeArea API

TABLET (iPad, 4:3):
- Scale UI elements 1.2x (larger touch targets)
- Increase spacing between elements
- Keep aspect ratio of panels

-------------------------------------------------------------------
TOUCH INTERACTION GUIDELINES
-------------------------------------------------------------------

1. Minimum touch size: 64x64 dp (recommended 80x80 dp)
2. Spacing between buttons: 16-20 px minimum
3. Feedback: Visual (scale/glow) + haptic (if enabled)
4. Response time: <100ms from touch to action
5. No precision taps (no small text links)
6. Gestures: Swipe for menus, pinch for zoom (future)

-------------------------------------------------------------------
ACCESSIBILITY FEATURES
-------------------------------------------------------------------

1. High contrast mode: +20% contrast on all text
2. Colorblind support: Icons + text labels (not just colors)
3. Text size: Scalable UI (settings option)
4. One-handed mode: Move joystick to right side (toggle)
5. Auto-battle: Essential for accessibility

-------------------------------------------------------------------
PERFORMANCE OPTIMIZATION
-------------------------------------------------------------------

1. UI Atlas: Combine all UI sprites into 2-3 atlases
2. Draw calls: Target <10 for UI (batch where possible)
3. Canvas: Use "Screen Space - Camera" for UI effects
4. Event system: Disable raycast on decorative elements
5. Updates: Only update changing elements (HP bar, cooldowns)
